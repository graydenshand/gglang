# Geometries

## Point

Required mappings:
- x
- y

Supported mappings:
- color
- size
- shape

Default statistic:
- identity

## Bar

Required mappings:
- x
- y

Supported mappings:
- color
- size
- fill

Default statistic:
- count

## Line

Required mappings:
- x
- y

Supported mappings:
- color
- size
- group
- linetype
